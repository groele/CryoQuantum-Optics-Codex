02 · 标准测试流程

测量前

  1. 定义样品坐标：晶轴、磁场方向、激光传播方向。
  2. 以标准样品校准波长、功率和偏振消光比。
  3. 记录物镜 NA、光栅、狭缝、积分时间、累计次数。
  4. 先做低功率功率依赖，排除加热、漂移和光致掺杂。

测量后

  1. 保留原始数据，不覆盖、不手动平滑。
  2. 背景、仪器响应和偏振通道响应分别校正。
  3. 报告拟合模型、初值、约束、置信区间和残差。
  4. 关键结论至少由重复位置或重复循环验证。

03 · PL 光致发光原理

光子能量高于带隙时产生电子—空穴对；载流子通过载流子—声子散射弛豫到带边，随后形成中性激子、带电激子或缺陷束缚态，并通过辐射或非辐射通道复合。

稳态速率模型

dN/dt = G − (k_r + k_nr + k_sc)N；稳态 N = G/k_tot；I_PL ∝ k_r N

功率指数 I∝Pα：α≈1 常见于未饱和自由激子；α<1 可能对应缺陷饱和、激子—激子湮灭或加热。

可提取信息

  • 峰位：带隙重整化、应变、介电屏蔽、磁场和 Stark 位移。
  • 线宽：无序、声子散射、寿命展宽和谱扩散。
  • 强度：吸收、量子效率、态占据与收集效率的乘积。
  • X⁰/X⁻：掺杂和电荷转移的相对指标，需结合门压或反射确认。

04 · Raman 散射原理

Raman 是二阶光—物质相互作用。入射光通过虚中间态与晶格振动交换能量：Stokes 产生一个声子，Anti-Stokes 湮灭一个已有声子。峰位反映晶格动力学，强度由 Raman 张量、偏振配置和共振条件决定。

偏振选择定则

I ∝ |e_sᵀ · R(mode) · e_i|²

ei 和 es 分别是入射与散射偏振矢量，R 是声子模 Raman 张量。旋转样品、半波片或分析器时，必须明确改变的是晶体坐标还是实验室偏振坐标。

温度计与加热诊断

I_AS/I_S ≈ C·exp(−ℏΩ/kBT)

Anti-Stokes/Stokes 比值可估算局部温度，但必须修正光谱仪响应、频率四次方因子和滤光片透过率。

05 · 反射、差分反射与吸收

二维材料吸收很弱，常以基底作为参考，通过差分反射增强激子共振。

ΔR/R = (R_sample − R_ref)/R_ref；在薄层和弱吸收近似下，ΔR/R 与吸收谱线形相关，但并非普适等于 absorbance。
反射信号强烈依赖 SiO₂ 厚度、入射 NA、偏振、焦点和参考区域。样品与参考必须在相同光路状态下测量。

06 · 偏振光路与校准

线偏振

DOLP = (Imax − Imin)/(Imax + Imin)

典型拟合：I(θ)=I₀+A cos[2(θ−θ₀)]。HWP 旋转 θ，理想线偏振方向旋转 2θ。

圆偏振

DOCP = (Iσ+ − Iσ−)/(Iσ+ + Iσ−)

QWP 零位、延迟误差、镜面反射和光谱仪偏振响应都会造成伪 DOCP。推荐以非偏振标准源得到通道校正系数。

07 · K/K′ 谷选择定则

单层 2H-TMD 缺少反演对称性，K 与 K′ 谷具有相反 Berry 曲率、轨道磁矩和圆偏振选择定则。具体 σ⁺/σ⁻ 符号必须与实验光传播方向和仪器定义统一。

08 · 四通道圆偏振测量

完整四通道能够区分激发端初始化不对称、探测端响应不对称与样品本征谷动力学。

通道物理含义主要用途
I++σ⁺ 激发，σ⁺ 探测同手性保持
I+-σ⁺ 激发，σ⁻ 探测谷翻转/交叉通道
I-+σ⁻ 激发，σ⁺ 探测反向交叉通道
I--σ⁻ 激发，σ⁻ 探测反向同手性保持

激发分辨 DOCP

Pσ+ = (I++ − I+-)/(I++ + I+-)
Pσ− = (I-- − I-+)/(I-- + I-+)

对称/反对称组合

P_even=(Pσ+ + Pσ−)/2；P_odd=(Pσ+ − Pσ−)/2

Peven 更敏感于与激发手性无关的磁场偏置；Podd 更敏感于光学初始化和手性选择性过程。

推荐顺序：++ → +- → -- → -+ → 重复 ++。首尾 ++ 的差异用于估计漂移；每个通道保持相同功率、积分时间和光斑位置。

09 · 谷 Zeeman 与磁场几何

Faraday geometry

B ∥ k。适合解析圆偏振发光、谷 Zeeman 劈裂和磁场诱导谷占据差。

ΔE = Eσ+ − Eσ− = g_eff μB B

Voigt geometry

B ⟂ k。可混合自旋态、激活暗激子或改变线偏振本征态，不能简单套用 Faraday 的 σ⁺/σ⁻ 图像。

μB = 5.7883818×10⁻² meV/T

若存在零场细结构 δ，可使用 |ΔE| = √[δ² + (gμBB)²]，而非强制线性过原点。

10 · 数据分析与误差传播

峰拟合

明确采用 Gaussian、Lorentzian、Voigt 或多峰约束。比较 AIC/BIC 与残差，不以视觉平滑代替模型选择。

偏振误差

DOCP 分母接近零时误差会发散。建议以重复测量或 Monte Carlo 传播背景、计数和校正系数的不确定度。

g 因子

同时拟合正负磁场，报告斜率标准误、95% CI、截距和 R²；峰配对规则必须固定。

σ²_P ≈ (∂P/∂I₊)²σ²₊ + (∂P/∂I₋)²σ²₋；其中 P=(I₊−I₋)/(I₊+I₋)
