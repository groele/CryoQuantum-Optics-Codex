# CryoQuantum-Optics-Codex

> **Version: 3.0.0**
>
> A single-file, fully offline and interactive handbook for low-temperature quantum-optical spectroscopy of two-dimensional materials.

## v3.0 Highlights

- Rebuilt as one self-contained HTML file with no external image or JavaScript dependency.
- Added publication-style inline SVG diagrams for:
  - confocal photoluminescence and exciton recombination;
  - Stokes / Anti-Stokes Raman scattering;
  - shared-waveplate polarization pathways;
  - K/K′ valley-selective optical transitions.
- Added detailed guidance for PL, Raman, differential reflectance, linear/circular polarization, four-channel valley measurements, valley Zeeman spectroscopy, fitting uncertainty and quality control.
- Added offline calculators for DOCP/DOLP, excitonic g-factor and photon energy.
- Added responsive dark/light themes and print/PDF layout.

## System Scope

The handbook is designed around a low-temperature micro-spectroscopy platform with representative capabilities of 1.65–300 K, ±9 T, 532 nm excitation, PL/Raman/reflectance measurements, polarization resolution and optional electrical control.

## Usage

Download `低温量子光学测试系统手册.html` and open it directly in a modern browser. All diagrams, styles and calculations are embedded in the file and work offline.

## Measurement Philosophy

Every physical conclusion should remain traceable to the optical state, sample coordinate system, excitation power, collection geometry, calibration record, fitting model and uncertainty definition.
