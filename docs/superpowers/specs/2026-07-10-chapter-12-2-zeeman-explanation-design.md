# 12.2 Zeeman 分裂与有效 g 因子：教学式重写设计

## 1. 目标读者与目标

目标读者已经能够阅读 PL 光谱并理解峰位、峰面积和偏振通道，但首次系统接触磁光光谱、Zeeman 分裂和有效 g 因子提取。

重写目标不是增加更多公式，而是建立一条完整认知链：

> 磁矩与磁场耦合 → 两个圆偏振通道峰位反向移动 → 定义能量差 → 对磁场拟合斜率 → 换算有效 g 因子 → 检查符号、残差和适用边界。

## 2. 章节定位

- 第 12.2 节承担“实验操作所需的最小理论”：读者读完即可理解测量对象、公式含义和提取步骤。
- 第 16 章继续承担“更深入的材料物理”：自旋、原子轨道、谷轨道、Berry 曲率、多体效应和不同激子态的 g 因子来源。
- 第 12.2 节通过链接提示读者进入第 16 章，不重复完整理论推导。

## 3. 内容结构

### 3.1 先给物理图像

用两段正文解释：

1. 具有磁矩的量子态在外磁场中会获得与磁矩方向有关的能量修正；相反磁矩对应相反能移。
2. 在 Faraday 几何下，K 与 K′ 谷通过相反圆偏振通道被选择性探测。外磁场解除时间反演相关态的能量简并，于是 σ+ 与 σ− 光谱峰发生相对移动。

措辞边界：不把 σ+ 永久等同于某一个固定能谷；对应关系取决于观察方向、磁场正方向和实验室圆偏振定义，必须由本手册的符号约定固定。

### 3.2 增加简洁能级—光谱示意

图分为三个阶段：

- B = 0：两个相关态在理想情况下能量简并。
- B > 0：两个态向相反方向移动，能量差为 ΔE。
- 光谱读出：分别从 σ+ 与 σ− 通道拟合峰位 Eσ+ 和 Eσ−。

图中明确标注“示意图，不代表实测数据”。不绘制没有来源的定量坐标或具体材料参数。

### 3.3 逐层引入公式

公式顺序保持为：

1. 定义观测量：

   ΔE(B) = Eσ+(B) − Eσ−(B)

2. 在线性响应区建立拟合模型：

   ΔE(B) = geff μB B + ΔE0

3. 由斜率 m = dΔE/dB 得到：

   geff = m / μB

4. 给出玻尔磁子：

   μB = 57.8838 μeV/T = 0.0578838 meV/T

每个公式后立即解释“测什么、单位是什么、斜率或截距表示什么”。

### 3.4 增加带单位的示例计算

采用明确标注为教学示例的数据：

- 假设 ΔE 对 B 的拟合斜率 m = −0.230 meV/T。
- geff = −0.230 / 0.0578838 ≈ −3.97。

示例用于说明单位换算和符号传递，不宣称对应某一真实样品。

### 3.5 增加符号约定说明

单独设置提示框，说明：

- 本手册定义 ΔE = Eσ+ − Eσ−。
- 交换两个通道的相减顺序，或反转磁场正方向，会改变拟合斜率与 geff 的符号。
- 因此报告 geff 时必须同时报告 ΔE 定义、磁场正方向、光传播方向和 σ± 定义。
- |geff| 表示分裂对磁场的响应强度；符号只有在约定完整时才具有可比较性。

### 3.6 将零场细结构改为特殊情况

现有公式

|ΔE(B)| = sqrt[δ² + (geff μB B)²]

保留但降级到“特殊情况”提示框，并说明：

- 它描述存在可分辨零场耦合或细结构能标 δ 时的两能级模型。
- 它不是所有自由 TMD 激子的通用公式。
- 使用前必须有零场谱、偏振依赖或独立证据支持 δ 的存在。
- 若数据在 B = 0 附近仍呈普通线性并且没有可分辨反交叉，不应强行使用该模型。

### 3.7 增加实验提取流程

用五步短流程连接理论与操作：

1. 在每个磁场点分别拟合 σ+、σ− 峰位。
2. 计算同一激子态的 ΔE(B)。
3. 绘制 ΔE–B 并检查线性区、残差和离群点。
4. 用斜率除以 μB 得到 geff，并传播斜率不确定度。
5. 报告截距、置信区间、通道定义和磁场扫描方向。

### 3.8 增加常见误判

- σ+ 与 σ− 强度差不是 Zeeman 能量分裂。
- 未分峰宽谱的重心移动不能自动替代各激子态峰位。
- 中性激子、三子、局域激子和极化子必须分别拟合与报告。
- 若 ΔE–B 存在明显非线性、磁滞或分段斜率，不能用单一线性 geff 覆盖全部磁场区间。
- 偏振串扰、窗口 Faraday 旋转、温度漂移和峰重叠会造成系统偏差。

## 5. 科学准确性边界

- geff 是特定光学跃迁、激子复合物、掺杂状态和测量约定下的有效量，不直接等同于单一电子自旋 g 因子。
- 线性公式只用于数据支持的线性响应区。
- ΔE0 可吸收零场残余分裂、拟合偏置或系统误差，但不能仅凭非零截距就宣称存在本征零场劈裂。
- 对中性激子、带电激子或极化子，不预设相同 geff；多体与最终态效应可能改变观测值。
