# 低温量子光学测试系统手册：功能框图与图文优化设计

## 1. 目标

在不改变手册 34 章总体结构、既有交互工具和科学结论边界的前提下，通过分层配置的高价值图表和文字精修，提高复杂流程的可理解性、章节之间的连贯性以及页面视觉完成度。

本轮坚持“按信息结构选图”：保留现有 6 个概念图，在真正存在流程、分支、层级、矩阵或版式关系的章节增加视觉表达；已有清晰参数表、诊断表或检查清单的章节不重复图示化。

## 2. 方案比较与选择

- 轻量方案：仅补 2–3 个图，改动小，但对校准、排障和决策章节的帮助有限。
- 中等密度方案：补 5 个核心流程图，并统一视觉语言、图注和相关导语；信息增益与页面长度较平衡。
- 分层增强方案：核心流程图之外，再补测量几何、通道矩阵、状态时间线、对照矩阵和排版示例；覆盖更完整，但必须严格控制单图复杂度。

经第二轮逐章审查，采用“分层增强方案”：核心流程图必须完成，物理与测量示意图只表达现有文字已经定义的关系，辅助矩阵和排版示例用于减少长列表，不新增虚构实验数据。

## 3. 一级：核心功能框图

### 3.1 标准快速使用流程

位置：第 4 章。

内容：设备状态确认 → 低功率定位 → 建立基准态 → 执行主扫描 → 插入参考点 → 数据备份与初检。

作用：把通用操作步骤与“开始测试前关键提醒”连接起来，突出未通过检查时不得进入下一步。

### 3.2 偏振校准闭环

位置：第 19 章。

内容：生成已知偏振态 → 样品面验证 → 检测端响应测量 → 建立校正矩阵 → 独立状态回验 → 应用到样品数据。

作用：强调校准不是一次测量，而是包含独立回验的闭环；区分激发端和检测端职责。

### 3.3 故障排查逻辑

位置：第 23 章。

内容：异常症状 → 回到基准态 → 判断是否可逆 → 光路/样品/仪器/分析/历史分类 → 单变量验证 → 记录处理结果。

作用：避免直接修改拟合参数，把排障顺序变成可执行流程。

### 3.4 实验设计决策链

位置：第 29 章。

内容：科学问题 → 待区分机制 → 最小观测量 → 必要控制变量 → 对照与重复 → 扫描顺序与停止条件。

作用：将“从科学问题反推测量矩阵”可视化，突出每增加一个维度都必须有明确用途。

### 3.5 数据质量与结论等级门槛

位置：第 30 章。

内容：原始数据完整性 → 校准通过 → 信噪比/拟合质量 → 重复性 → 对照排伪 → 探索趋势/现象观察/机制支持。

作用：明确质量检查是逐级门槛，结论强度不能超过证据等级。

## 4. 二级：物理关系与测量几何示意

### 4.1 Stokes / anti-Stokes 能量交换示意

位置：第 7 章。

形式：激光光子、声子吸收/发射和散射光子三层能量关系图，并在旁侧标出 anti-Stokes 通道依赖热布居、两侧强度比较必须做仪器响应校正。

作用：帮助读者区分 Raman 位移、光子能量和声子能量，避免把 Stokes/anti-Stokes 强度比直接当作无校正温度计。

### 4.2 三类线偏振测量几何对照

位置：第 9 章。

形式：激发偏振依赖、发射偏振依赖、共/交偏振三栏对照图；固定元件使用灰色，旋转元件使用强调色，输出物理量放在每栏底部。

作用：直观区分“转哪一侧、固定哪一侧、得到什么信息”，减少机械角、样品面偏振角和检测分析角混用。

### 4.3 四通道圆偏振矩阵与正交分解

位置：第 10 章。

形式：先用 2×2 矩阵定位 I++、I+-、I-+、I--，再以箭头连接到总亮度、激发不对称、检测不对称和手性关联四个正交量；旁侧标明“原始强度 → 效率校正 → 分峰积分 → 派生量”。

作用：把多个公式归入同一个数据结构，明确 DOCP、保持率、翻转率和正交分解并非相互替代。

### 4.4 电学写入—读取—保持—耐久时间线

位置：第 11 章。

形式：门电压与光学读取的双轨时间线，区分动态扫描、写入后零偏读取、保持测试和循环耐久；在写入与读取之间明确回到 0 V。

作用：区分即时静电调控与非易失记忆证据，避免把有偏压测得的变化误称为零偏保持。

### 4.5 Faraday / Voigt 几何与奇偶分解

位置：第 12 章。

形式：左侧对照磁场 B、光传播 k 和样品面的相对方向；右侧用 +B、0、−B 三点说明奇分量与偶分量的组合关系。

作用：把测量几何和数据对称性放在同一图中，帮助读者判断谷 Zeeman、磁化响应、加热和光路伪影应如何区分。

### 4.6 重复性层级与不确定度来源

位置：第 20 章。

形式：同一点 → 不同位置 → 不同循环 → 不同器件 → 不同批次的层级图；下方并列拟合不确定度、短时噪声、系统误差和样品差异。

作用：明确不同重复层级回答的问题不同，软件拟合标准误不能代替器件或批次重复性。

### 4.7 低温与磁场操作状态流程

位置：第 22 章。

形式：室温参考 → 降温 → 温度稳定 → 低功率重新对焦 → 低温基准谱 → 本地控制确认 → 磁体加热器状态确认 → 扫场 → 回零与状态记录；设置明确的停止/复查节点。

作用：把设备状态、安全动作和数据基准串联起来；不添加设备厂商规程之外的新参数或自动控制指令。

## 5. 三级：辅助矩阵与论文表达示例

### 5.1 典型实验模板选择矩阵

位置：第 21 章。

形式：实验问题 × 最小扫描维度 × 必需对照 × 推荐输出的紧凑矩阵。

作用：让读者在四通道、磁场、栅压、铁电、温度和线偏振模板之间快速选择，不重复每节的具体操作步骤。

### 5.2 高级分析方法选择矩阵

位置：第 24 章。

形式：二维参数图、联合拟合、PCA/聚类、Jones/Mueller 矩阵、时间分辨方法与适用问题、最低数据条件、主要风险对应表。

作用：强调高级方法的适用边界，避免因图像平滑或模型复杂而产生过强结论。

### 5.3 安全风险—控制—停止条件矩阵

位置：第 26 章。

形式：激光、磁场、低温/真空、电学、数据完整性五类风险，对应主要后果、预防动作和必须停止的状态。

作用：把安全内容变成可执行判定；不使用虚构概率或风险分数。

### 5.4 字体与学术符号对照样张

位置：第 32 章。

形式：正确/错误两栏，实际展示变量斜体、单位正体、数字与单位空格、上下标、材料名和希腊字母的组合。

作用：把抽象排版规则变成可直接模仿的样张。

### 5.5 配色、线型与色盲可读性样张

位置：第 33 章。

形式：离散色、有序色、发散色各一组色带，并展示颜色与实线/虚线/点型的组合；附黑白打印缩略示例。

作用：避免只凭颜色区分通道，并让配色建议能够直接复用到论文图。

### 5.6 多面板证据链版式

位置：第 34 章。

形式：在现有低效/有效版式对照基础上，增加“测量几何 → 原始数据 → 定量提取 → 统计/对照 → 机制边界”的面板角色图，并标出主次面积和阅读顺序。

作用：让多面板拼图承担论证结构，而不只是对齐与美化。

## 6. 明确不新增图示的章节

- 第 3 章已有分阶段检查条目和表格，优先改善分组与视觉层级，不再另加流程图。
- 第 6 章已有 PL 红移/蓝移诊断矩阵，保留表格比再画决策树更便于查阅。
- 第 13 章包含温度、功率和时间的多模型边界，优先强化小标题、公式说明和对照表，不把不同模型压成一张容易误读的总图。
- 第 17 章以材料参数速查为主，表格是最合适的信息载体。
- 第 27 章已有可勾选检查清单，重点优化阶段分隔和打印效果，不重复制作流程图。
- 第 28 章已有参数边界表，补充表头和单位说明即可。

## 7. 视觉设计

- 沿用现有蓝、青、橙、绿语义色，不引入新的品牌色体系。
- 将框图分为输入、过程、校验、风险/停止、输出五类节点；关键回验或停止节点使用强调边框和文字标签。
- 优先采用 HTML/CSS 节点与连接符，避免固定填充色 SVG 在暗色模式中出现底色和文字不协调；现有 SVG 保留但补充主题适配。
- 桌面端以横向流程为主；窄屏自动改为纵向堆叠，连接方向随布局变化。
- 图注统一回答“该图如何使用”和“最容易误判什么”，不重复图中节点文字。
- 打印模式保持黑白可辨，框线、编号和节点标签不能只依赖颜色传意。
- 所有物理关系图和示意曲线明确标注“示意图，不代表实测数据”；不绘制没有来源的定量坐标、误差条或阈值。
- 图、表和正文使用连续且一致的命名方式，例如“测量几何”“处理流程”“判定门槛”“排版示例”，避免所有视觉元素都笼统称为“功能框图”。

## 8. 文字优化边界

- 精修新增框图前后的章节导语，使其按“为什么重要—如何执行—如何判定”展开。
- 消除相邻段落重复，统一“样品面”“检测端”“基准态”“回验”“质量门槛”等术语。
- 不新增未经手册现有内容支持的设备参数、物理结论或经验阈值。
- 对测量、拟合、校准、推断和机制结论使用不同强度的表述，避免把示意图当作实测证据。

## 9. 交互、可访问性与验证

- 框图提供清晰的章节标题、序号、文字标签与辅助说明；不依赖悬停才能读取内容。
- 检查浅色、暗色、桌面和窄屏布局；确认无横向溢出、文字截断或连接线错位。
- 运行现有结构验证脚本，并扩展校验标记以覆盖新增核心框图、测量示意、辅助矩阵和排版样张。
- 核验目录跳转、主题切换、搜索、打印隐藏规则和既有分析工具未受影响。
- 浏览器检查图表在 1440 px、1024 px 和手机宽度下的阅读顺序；移动端不允许依赖水平拖动才能理解主流程。

## 10. 完成标准

1. 5 个核心功能框图、7 个物理/测量示意、3 个辅助矩阵和 3 个排版样张或增强示例均位于对应章节，并各自承担明确的信息表达任务。
2. 新增图示在浅色、暗色、打印和移动端均清晰可读；颜色不是唯一编码手段。
3. 所有示意性内容均明确标注，不与实测数据、拟合结果或设备实测参数混淆。
4. 相关章节导语、图注和术语统一，无明显重复、逻辑跳步或过度承诺。
5. 现有 34 章、导航、公式、工具和本地离线能力保持正常。
6. 自动结构验证与浏览器关键路径核验均通过。
